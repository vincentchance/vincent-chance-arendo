Api

- download it in github or clone it
- open xampp on search bar in desktop if installed.
- activate apache and mysql in xampp, then click admin, automatically myphpadmin opens in your browser.
- make a new database named product.db in myphp
- open data with vscode or open cmd in your document search path where is the api is located.
- write node index.js and enter with your console
- rest api is recommended if you want to test the api in your vscode. 
- data table after api testing should be appeared in database myphpadmin.

Note: You can use request.rest for api testing with rest client extension in vscode.
